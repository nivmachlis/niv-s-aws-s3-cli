<h1>niv-cli</h1>  
<h3>This is a CLI tool for the following S3 operations:</h3> <br>
1. List objects by bucket name <br>
2. Put object by bucket name and object path <br>
3. Delete object by bucket name and object path <br>

<h3>Requirements:</h3>
1. python 3  <br>
2. aws-cli  <br>
3. boto3  <br>
